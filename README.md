# YousifWebsite

موقع شخصي بسيط يعرض حياة يوسف مع صفحات لمقاطع الفيديو والصور والحسابات والأصدقاء والمواقع المفضلة.
تتوسط الصفحة الرئيسية صورة مربعة ليوسف مع وصف قصير يليه معرض لصور الألعاب التي يحبها.
التصميم يعتمد على **دارك ثيم** مع تأثيرات تفاعلية وهوفر.
تم تكبير الخطوط وإتاحة إضافة وتعديل وحذف البيانات في الصفحات المختلفة باستخدام LocalStorage.
عند فتح الموقع يتم تحويلك إلى صفحة تسجيل الدخول التي تقبل فقط المستخدم **Yousif** وكلمة المرور **1234**.
